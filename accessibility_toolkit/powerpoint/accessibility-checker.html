Different types of issues

Errors

Errors are issues which are difficult, or impossible for someone with a disability to understand. These need to be resolved before we can upload the document to the website.

Warnings

Warnings are issues that could cause problems for someone with a disability. You should attempt to fix all of these issues, however if you are sure that it won't cause any issues for anyone, they can be left.

Tips

Tips are issues where someone with a disability will be able to understand the content, however it could be changed to make it easier for them.

Intelligent Services

Intelligent Services is where Microsoft Word has attempted to auto-generate alt text for your images and objects. They should be reviewed by you to make sure that they're accurate and descriptive
